# Проект: Лучше чем вчера (AI Advent with Love #3)

## Контекст
Бэкенд для трекинга физической активности (прогулки/пробежки). Мобильного приложения пока нет, фокус на серверной логике.

## Логика работы
- **Сессия:** Кнопка "Старт" активирует запись данных каждые 10 минут в течение 2 часов.
- **Пауза:** После 2 часов запись останавливается до ручного запуска "Продолжить".
- **Данные:** Координаты (Yandex Maps), шаги/дистанция (гироскоп), время, скорость.
- **Аналитика:** Выявление цикличности маршрутов. Сравнение текущей сессии с историей за 3 дня.
- **AI Агент:** Генерирует мотивирующее сообщение на основе сравнения прогресса.

## Специфические требования к коду
1. **Структура пакетов:** `com.example.betterthanYesterday.{controller, service, repository, entity, dto, config, security}`.
2. **Keycloak:** Настройка через `application.yaml`, права доступа через `@PreAuthorize`.
3. **Сущности:**
   - `WalkSession`: `startTime`, `endTime`, `status` (ACTIVE, PAUSED, FINISHED), `totalDistance`.
   - `WalkPoint`: `lat`, `lon`, `timestamp`, `stepCount` (связь ManyToOne с сессией).
4. **Планировщик:** Запись данных каждые 10 мин реализовывать через `@Scheduled`.
5. **AI Интеграция:** Сервис отправляет JSON (сессия + история 3 дней) в OpenRouter. Ожидает ответ строго в формате: `{ "isBetter": boolean, "message": "string", "improvementPercent": number }`.
6. **Обработка ошибок:** Глобальный обработчик через `@ControllerAdvice`.
